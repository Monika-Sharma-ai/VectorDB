NSW — Navigable Small World

Build a graph where each point links to a few neighbors. Walk greedily toward the query.

Intuition Like asking a stranger "who do you know that's closer?" — random insertion creates both short hops (precision) and long hops (jumps), the small-world property.

BuildInsert points one by one; connect to M closest already in graph
PropertyMix of short + long edges emerges naturally
QueryStart anywhere, hop to closer neighbor each step
StopWhen no neighbor is closer (local minimum)

Weakness: can get stuck in local minima on a single layer → fixed by HNSW.

HNSW — Hierarchical Navigable Small World ⭐

Stack multiple NSW graphs like a skip-list. Top = sparse long jumps; bottom = dense precision.

Why hierarchy Prevents getting stuck in local minima — top layers do the long jumps for free.

  • Each point gets a random level l (geometric distribution → few high-level nodes)
  • Layer 0 contains every point; upper layers are progressively sparser
  • Search descends top → bottom, refining at each layer

Knobs: M (neighbors per node), efConstruction (build quality), efSearch (query recall vs latency).

HNSW Search — Top-Down Greedy Descent

Coarse jumps at the top, fine beam search at the bottom.

Algorithm Enter at top layer's entry point. Greedy walk → drop one layer → repeat. At Layer 0, beam-search with efSearch candidates.

L2Greedy from entry → local min
L1Continue from that node → refine
L0Beam search with efSearch=200
OutTop-K nearest neighbors

Complexity: ~O(log N) per query · Recall: 95–99%+ at well-tuned ef.

DiskANN — Billion-Scale Graphs on SSD

Keep tiny PQ codes in RAM for fast distance estimates; full vectors + Vamana graph live on SSD.

Why HNSW + full vectors at 1B scale = TBs of RAM. DiskANN keeps RAM small and lets the SSD do the storage work.

  • Vamana graph on SSD — HNSW-style navigability, single layer
  • PQ-compressed vectors in RAM — fast approximate distance
  • Only when a candidate looks promising does it fetch the full vector from disk

Result: billion-vector ANN on a single machine with ~64 GB RAM.
